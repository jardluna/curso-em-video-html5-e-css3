Trabalhando Com Listas

Listas Ordenadas

Usamos a tag <li></li> de forma indentada a tag <ol></ol> para criar Listas Ordenadas. Obs: A partir do HTML5, o fechamento da tag </li> é opcional.

Exemplo de código de Lista Ordenada: 

        
<body>
    <ol>
        <li>Acordar</li>
        <li>Ligar para o João</li>
        <li>Tomar café</li>
        <li>Escovar os dentes</li>
    </ol>
</body>

Exemplo de Lista Ordenada:

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes

As Listas Ordenadas tem como padrão marcadores numerados, porém podemos mudar os marcadores utilizando o parâmetro type dentro da tag <ol>. Os marcadores são: type="1" (lista numerada - padrão), type="A" (lista alfabética em maiúscula), type="a" (lista alfabética em minúscula), type="I" (lista em algarismo romano em maiúscula), type="i" (lista em algarismo romano em minúscula).

Exemplo de código de Lista Ordenada com parâmetro type: 

        
<body>
    <ol type="a">
        <li>Acordar</li>
        <li>Escovar os dentes'</li>
        <li>Fazer exercício</li>
        <li>Ir estudar</li>
    </ol>
</body>

Exemplo de Lista Ordenada com parâmetro type:

  1. Acordar
  2. Escovar os dentes
  3. Fazer exercício
  4. Ir estudar

Podemos também determinar qual o valor inical do marcador, utilizando o parâmetro start dentro da tag <ol>.

Exemplo de código de Lista Ordenada com parâmetro start: 

        
<body>
    <ol start="4">
        <li>Acordar</li>
        <li>Mexer no celuar</li>
        <li>Ir ao mercado</li>
        <li>Assistir TV</li>
    </ol>
</body>

Exemplo de Lista Ordenada com parâmetro start:

  1. Acordar
  2. Mexer no celular
  3. Ir ao mercado
  4. Assistir TV

E por fim, exemplo de código e de Lista Ordenada com parâmetro type e start: 

        
<body>
    <ol type="I" start="3">
        <li>Acordar</li>
        <li>Tomar banho</li>
        <li>Passear com o cachorro</li>
        <li>Limpar a casa</li>
    </ol>
</body>
  1. Acordar
  2. Tomar banho
  3. Passear com o cachorro
  4. Limpar a casa

Listas Não Ordenadas

Usamos a tag <li></li> de forma indentada a tag <ul></ul> para criar Listas Não Ordenadas. Obs: Novamente, a partir do HTML5, o fechamento da tag </li> é opcional.

Exemplo de código de Lista Não Ordenada: 

        
<body>
    <ul>
        <li>Matemática</li>
        <li>Português</li>
        <li>História</li>
        <li>Biologia</li>
    </ul>
</body>

Exemplo de Lista Não Ordenada:

  • Matemática
  • Português
  • História
  • Biologia

As Listas Não Ordenadas tem como padrão marcadores de bolinha, porém também podemos mudar os marcadores utilizando o parâmetro type dentro da tag <ul>. Os marcadores são: type="disc" (uma bola totalmente preta - padrão), type="circle" (uma bola preta sem preenchimento), type="square" (um quadrado totalmente preto).

Exemplo de código de Lista Ordenada com parâmetro type: 

        
<body>
    <ul type="circle">
        <li>Educação Física</li>
        <li>Geografia</li>
        <li>Química</li>
        <li>Artes</li>
    </ul>
</body>

Exemplo de Lista Não Ordenada com parâmetro type:

  • Educação Física
  • Geografia
  • Química
  • Artes
