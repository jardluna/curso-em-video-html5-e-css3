Formatação de Textos

Principais Formatações

Negrito / Destaque

 → Nessa frase temos um termo em Negrito usando a tag <b>exemplo de texto</b> (não possui semântica).

 → Nessa frase temos um termo em Destaque usando a tag <strong>exemplo de texto</strong> (possui semântica).

Itálico / Ênfase

 → Nessa frase temos um termo em Itálico usando a tag <i>exemplo de texto</i> (não possui semântica).

 → Nessa frase temos um termo em Ênfase usando a tag <em>exemplo de texto</em> (possui semântica).

Texto Marcado

 → Nessa frase temos um texto Marcado usando a tag <mark>exemplo de texto</mark>.

 → E nessa frase, o texto Marcado não herda a cor do texto acima.

Texto Grande e Pequeno

 → Nessa frase temos um texto Grande e um texto Pequeno usando as tags
<big>exemplo de texto</big> e <small>exemplo de texto<small> respectivamente.

Texto Deletado

 → Podemos marcar um texto como Excluido para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

 → Podemos marcar um texto como Inserido para dar ênfase e indicar que ele foi adicionado depois (possui semântica).

 → Existe também o Sublinhado com a tag <u>exemplo de texto</u> (não possui semântica).

Texto Sobrescrito

Para inserir algo acima do texo, como x20 + 3, usamos a tag <sup>exemplo de texto</sup>.

Texto Subescrito

Para inserir algo abaixo do texto, como H2O, usamos a tag <sub>exemplo de texto</sub>.
